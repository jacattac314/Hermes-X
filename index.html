🛠️ Local-Only Installation Guide

Follow these 7 steps to configure a fully sandboxed, highly-capable local environment on your Mac.

1 Install Ollama
⬜ Mark Done

Download and install the background daemon to run open-source models locally on macOS.

Terminal - brew
$ brew install ollama
Terminal - serve
$ ollama serve # Starts Ollama in the background
2 Pull Qwen Model
⬜ Mark Done

Pull a powerful local reasoning LLM. Use 32b for high quality or 14b if resources are limited.

Terminal - qwen3:32b
$ ollama pull qwen3:32b # Recommended Model
Terminal - qwen3:14b
$ ollama pull qwen3:14b # Lighter Fallback Model
3 Install Hermes Agent
⬜ Mark Done

Run the Nous Research Hermes Agent one-line shell installer, reload, and verify the diagnostics.

Terminal - bash install
$ curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
Terminal - reload & diagnose
$ source ~/.zshrc && hermes doctor
4 Configure Hermes model
⬜ Mark Done

Point the Hermes Agent at your local Ollama server node.

Terminal - config model
$ hermes model

Choose the following fields in the prompt:
• Provider: Custom / OpenAI-compatible / Ollama
• Base URL: http://127.0.0.1:11434/v1
• API Key: ollama
• Model: qwen3:32b (or qwen3:14b)

5 Test Local TUI
⬜ Mark Done

Launch the Hermes terminal user interface and test it with a local prompt.

Terminal - run tui
$ hermes --tui
Validation Prompt
"You are running locally through Ollama. Tell me what model you are using, then list my current directory. Do not edit files."
6 Launch Local Dashboard
⬜ Mark Done

Recent Hermes releases include a beautiful built-in web gateway interface for managing Hermes locally.

Terminal - gateway
$ hermes gateway run
🌐 Open Local Gateway (localhost:8642)
7 Enable Real-Time Browser Control
⬜ Mark Done

Let Hermes control and interact with your actual, visible Google Chrome browser window on your Mac using Chrome DevTools Protocol (CDP).

Terminal - launch chrome
$ /Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome --remote-debugging-port=9222 --user-data-dir="/Users/jack/.hermes/hermes2-chrome-profile"
Terminal - connect hermes
> /browser connect # Type this in active Hermes chat or TUI
